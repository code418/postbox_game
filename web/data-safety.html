The Postbox Game – Data Safety

What data the App collects and how it is used.

Summary: The App collects your location (while in use only), your account display name and email, and gameplay activity. Data is encrypted in transit. You can request deletion of your account and data at any time. We do not sell your data.

Data Collected

Data type What specifically Why Required?
Precise location GPS coordinates while using the App Find nearby postboxes; validate claims Required
Name Display name (chosen by you or from Google) Shown on leaderboards and to friends Required
Email address Email used to sign in Account authentication Required
App interactions Postbox claims, scores, streaks, leaderboard entries Core gameplay features Required
Crash logs Anonymous crash reports and stack traces Bug fixing and App stability Optional
App performance Anonymous performance traces (load times, etc.) Performance improvements Optional

Data Sharing

Question Answer
Is data shared with third parties? No – data is only processed by Google Firebase (our infrastructure provider)
Is data sold? No
Is data used for advertising? No
Is data used to track users across other apps? No

Security Practices

Practice Status
Data encrypted in transit Yes – HTTPS / TLS for all network requests
Data encrypted at rest Yes – Google Firebase encrypts data at rest by default
You can request data deletion Yes – contact us to delete your account and all associated data

Location Access

The App requests precise location permission. Location is accessed only while the App is in the foreground (i.e. not in the background). It is used solely to find postboxes near you and to validate that you are close enough to claim one. Location coordinates are not stored after a claim check completes.

Account Deletion

You can request deletion of your account and all associated personal data by contacting us at richard@agilepixel.io. We will process deletion requests within 30 days.
